Image To Philips Hue

The end goal of this project is to send an image to the Philips Hue lighting system, before this can be done the lighting system must be configured.

  • Add User

1. Configure the Bridge

Configure the bridge before changing the lighting. The default value for the bridge IP address is change to the IP address of your bridge before moving to the next step (source Developers MeetHue - Configuration).

2. Add User

Configure the bridge before changing the lighting. This function adds the user to the bridge. Configure with a new username. (source Developers MeetHue - Configuration). Add a new user

6. Change All Colors
